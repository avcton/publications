  1. Digital Image
    An two dimensional array of Function, F(x, y) where x and y represent coordinates and function F provides the intensity of image at that point When x, y and amplitude values of F are finite, we call it a digital image
    Pixel
    A digital image is composed of a number of finite elements. These elements are located in specific locations (co-ordinates) and have values. These elements which make up the image are called the picture element, image element, pel or pixel.
    Resolution
    Amount of detail or information present in a image
    Measured In
    Pixel Resolution
    Number of pixels on the screen, either vertically, horizontally or both
    1920 (Width - Number of Pixels) x 1080 (Height - Number of Pixels)
    Spatial Resolution
    It refers to the number of pixels per unit of physical distance in the image. For example, if an image has a spatial resolution of 300 pixels per inch (PPI), it means there are about 17 pixels horizontally and 17 pixels vertically within one inch of the image. Higher spatial resolution results in a crisper and more detailed image.
    Measured In
    Dots Per Inch (DPI)
    For Monitors, Printers etc.
    You have a 640 x 480 image
    You want to print it at 200 dpi (dots or pixels per inch).
    What will be the size of the printed image?
    640 / 200 = 3.2 and
    480 / 200 = 2.4
    So, the size of the printed image is 3.2" x 2.4"
    Lines Per Inch (LPI)
    For Laser Printers
    Pixel Per Inch (PPI)
    For Tablets, Mobile Phones
    Pixel Resolution2 can be also be converted to Spatial Resolution using the size of the device

    Intensity Resolution
    Intensity / grey level resolution can be defined as the smallest identifiable change in intensity level.
    In digital systems, intensity resolution is often quantified in bits. For example, in a digital image, an 8-bit intensity resolution allows for 256 different levels of intensity (2^8), ranging from 0 (minimum intensity) to 255 (maximum intensity). A higher bit depth, such as 12 bits or 16 bits, provides a greater number of intensity levels and thus a higher intensity resolution.
    Spectral Resolution
    Spectral resolution, in the context of remote sensing and spectroscopy, refers to the ability to distinguish and measure different wavelengths or bands within the electromagnetic spectrum. It is a measure of the level of detail or the number of spectral bands that can be differentiated in an image or a data set
    Aspect Ratio
    Expresses the relationship between width and height of the image
    For example, an image of 1920 x 1080 size, has an aspect ratio of 16:9 With 1920 as the width and 1080 the height

    307,200 x 8 = 2,457,600 Bits = 307,200 B = 300 KB
    In practice, the most commonly used bit depths for grayscale images are 8-bit, 16-bit, and occasionally 32-bit (floating-point representation). These bit depths provide sufficient levels of grey to represent a wide range of shades and details in grayscale images.

  4. Histogram Processing
    Histogram processing is a technique used in image processing to analyse and manipulate the distribution of pixel intensities in an image. It involves examining the histogram, which is a graphical representation of the frequency of occurrence of different intensity values within an image
    Methods
    Histogram Equalisation
    Normalising the histogram involves redistributing the intensity values of the image to span the entire available range, typically from 0 to 255 in an 8-bit grayscale image
    Calculate total number of Grey Levels (L) - in correspondence with max intensity
    Find the max intensity value
    Check in how many bits can it be represented
    That is your Bit Depth, so use to get your levels
    i.e. For 5, it is represented in 3 bits, so we have grey levels
    Calculate the total number of pixels / elements (N)
    Find frequency of each grey level
    Find probability of each grey level by dividing its frequency by N
    Calculate CDF for each grey level
    Multiply CDF by (L-1)
    Round Off to get the equalised Histogram Levels
    ​ ​
    Histogram Matching
    Histogram matching, also known as histogram specification or histogram equalisation matching, is a technique used to transform the histogram of an image to match a desired histogram. It is commonly employed for image enhancement and to adjust the appearance of an image based on a reference or target histogram
  5. Spatial Filtering
    Kernel
    AKA Filter, Mask, Template, Window
    Kernel is an array whose size defines the neighbourhood of operation to be performed to an Digital Image1
    Linear Spatial Filtering
    A linear spatial filter performs a sum-of-products operation between an image and the kernel
    ​ ​
    Put the kernel on every pixel in the image in such as way that the centre element of the kernel comes on top of that pixel. Then we multiply each kernel and pixel element in the neighbourhood and sum them in the end to place that final value in the position of the centre pixel.
    Spatial Correlation
    Correlation consists of moving the centre of a kernel over an image, and computing the sum of products at each location or simply applying the filter.
    Spatial Convolution
    Correlation consists of rotating the kernel by 180° and then moving the centre of the kernel over an image, and computing the sum of products at each location or simply applying the filter.

    Term Frequency-Inverse Document Frequency (TF-IDF)
    TF-IDF is a numerical statistic that is intended to reflect how important a word is to a document in a collection or corpus
    The tf–idf value increases proportionally to the number of times a word appears in the document and is offset by the number of documents in the corpus that contain the word.
    It is established as follow:
    Count the number of times each word appears in a document
    Calculate the frequency that each word appears in a document out of all the words in the document
    Term Frequency
    Term frequency (TF) shows how frequently an expression (term, word) occurs in a document
    Inverse Document Frequency
    IDF is a measure of how much information the word provides, i.e., if it’s common or rare across all documents
    It is the logarithmically scaled inverse fraction of the documents that contain the word
    TF-IDF gives larger values for less frequent words in the document corpus.
    TF-IDF value is high when both IDF and TF values are high i.e the word is rare in the whole documents but frequent in a document.

    Receptive Field
    In CNNs, neurons are organized in a hierarchical manner into multiple layers, including convolutional layers. In these layers, neurons are arranged in a grid-like structure, and each neuron is only connected to a local region of the input data, known as the receptive field. The receptive field in CNNs is determined by the size of the convolutional kernel (also called filter) applied to the input. As the network goes deeper, the receptive field of neurons in higher layers expands to cover larger areas of the input.
    The concept of receptive fields is crucial in CNNs because it enables the network to capture hierarchical and spatially local features from the input data. Neurons in earlier layers have smaller receptive fields and capture low-level features like edges and textures, while neurons in deeper layers have larger receptive fields and capture high-level features and global patterns.

  15. RNN
    More complex than ANN and CNN
    Recurrent neural networks (RNNs) are the state of the art algorithm for sequential data and are used by Apple’s Siri and Google’s voice search. It is the first algorithm that remembers its input, due to an internal memory, which makes it perfectly suited for machine learning problems that involve sequential data.
    It is one of the algorithms behind the scenes of the amazing achievements seen in deep learning over the past few years.
    Bidirectional RNNs also exist that keep track of the next element in the sequence
    Applications
    Speech Recognition
    Music Generation
    Sentiment Analysis (Multi-Class)
    DNA Sequence Analysis
    Video Activity Recognition
    Name Entity Recognition

    Dynamic Signals

  17. Deep Learning
    AKA Deep Neural Networks
    When we talk about deep learning, what we mean by “deep” is the depth of layers and nodes in a neural network. Thus, a neural network consisting of more than three layers (including input and output) is considered a deep learning algorithm.
    Deep Learning and Machine Learning18
    There’s the fact that deep learning models require far less human intervention than their regular machine-learning counterparts. If we take a self-driving car AI as an example, we could say that its ability to read road signs would be influenced by a software engineer’s manual intervention, also known as feature extraction. 
    This means that they would classify and sort images before feeding them through the neural network input layer, check whether they got the desired output, and adjust the algorithm accordingly if they didn’t. 
    On the other hand, with deep learning models, the feature extraction happens automatically, and the algorithm learns from its errors instead of relying on a software engineer to adjust and perform the feature extraction manually.
    Deep Learning and Neural Networks12
    A deep learning network has many distinct layers, making it more sophisticated than a neural network.
    A deep learning system performs tasks efficiently and effectively, whereas a neural network performs jobs slightly less efficiently than a deep learning system.
    A deep learning unit's main components are an ample power supply, a GPU, and a large RAM. In contrast, a neural network's main components are neurons, learning rate, connections, propagation functions, and weight.
    Due to their complexity, deep learning networks need significant training time, whereas neural networks require little training.
